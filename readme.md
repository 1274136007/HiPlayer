**HiPlay** 是一款精致小巧的音乐播放器。
其主要功能如下：
添加本地歌曲到播放列表；
自动通过网络获取专辑图片和歌词；
歌词桌面显示，且带有歌词遮罩效果；
Mini模式显示。
